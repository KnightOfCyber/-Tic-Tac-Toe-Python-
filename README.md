# Tic Tac Toe Game in Python

This is a simple console-based Tic Tac Toe game written in Python.
It is a two-player game played directly in the terminal.

---

## About the project

Players take turns entering positions from 1 to 9.
The program checks for a win or draw after each move.

The goal of this project is to practice logic building and condition checking.

---

## How to run

```bash
python tic_tac_toe.py

How to play

Player X goes first

Enter a number between 1 and 9

First player to get three in a row wins
